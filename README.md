# 롤챔추천

롤 전적검색과 전적 데이터 기반으로 챔피언을 추천해주는 OpenAI 활용 서비스입니다.

## 핵심 기능

- 롤 전적 조회
- 전적 데이터 기반 챔피언 추천

## 기술 스택

- SwiftUI
- Combine
- Swift Concurrency

## API

- Riot API
- OpenAI API
